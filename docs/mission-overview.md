# Overview
This document describes the mission objectives and high-level scope of the CubeSat project.

Misi CubeSat difokuskan pada pengembangan sistem satelit skala kecil yang mampu:
- Beroperasi secara mandiri
- Mengelola daya dari sumber energi terbatas
- Mengirimkan data telemetri secara berkala
- Menjalankan payload sesuai tujuan misi

Dokumentasi ini disusun sebagai acuan konseptual dan tidak menggambarkan implementasi teknis secara rinci.
