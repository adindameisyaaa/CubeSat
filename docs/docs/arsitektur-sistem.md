# Arsitektur Sistem CubeSat
Sistem CubeSat terdiri dari beberapa subsistem utama yang saling terintegrasi untuk mendukung operasi satelit secara keseluruhan.

Secara umum, arsitektur sistem CubeSat mencakup:
- Sistem pengelolaan daya
- Sistem kendali dan pemrosesan data
- Sistem komunikasi
- Sistem struktur dan mekanik
- Payload

Setiap subsistem memiliki fungsi spesifik namun bekerja secara terkoordinasi untuk memastikan misi dapat berjalan dengan baik.
